# RPG Calculator Platform

  A premium, full-featured RPG Calculator Factory Platform for tabletop RPGs (D&D 5e, Pathfinder 2e).

  ## Features

  - 🎲 **30 Interactive Calculators** — 8 custom tools + 22 formula-driven calculators
  - 📚 **50 RPG Guides** — Ability scores, class guides, combat mechanics, and more
  - 🏆 **221+ Character Builds** — Pre-optimized builds for D&D 5e and Pathfinder 2e
  - 🧙 **13 Class Pages** — Stat guides, optimal point buy allocations, and build tips
  - 🔍 **Global Search** — Search across all builds, guides, calculators, and classes
  - 🔗 **Shareable Links** — Share your character builds with a URL
  - 💾 **Build Export** — Export and import character builds
  - ⚡ **100% Client-Side** — No backend required, runs entirely in the browser

  ## Tech Stack

  - React 18 + TypeScript
  - Vite
  - Tailwind CSS + shadcn/ui
  - Framer Motion
  - Recharts
  - Three.js (3D effects)

  ## Getting Started

  ```bash
  pnpm install
  pnpm --filter @workspace/point-buy-calculator run dev
  ```
